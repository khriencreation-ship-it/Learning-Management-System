# Khrien Academy LMS

A guided Learning Management System for structured, time-bound, mentor-led education.

## Features

- **Super Admin Dashboard** - Platform governance and management
- **Tutor Dashboard** - Teaching and mentorship tools
- **Student Dashboard** - Guided learning experience
- **Content Locking** - Time-bound lesson unlocking
- **Live Sessions** - Weekly mentor-led classes
- **Messaging System** - Cohort-based communication

## Tech Stack

- **Frontend**: Next.js 16 with TypeScript
- **Styling**: Tailwind CSS
- **Database**: MongoDB (planned)
- **Storage**: Supabase (planned)
- **Authentication**: JWT (planned)

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
